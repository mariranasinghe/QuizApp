The application is a full suite of tools, not just a single quiz app.

1. **Quiz Game (Default View)**: The app initiates by loading the main Quiz Game interface. From here, users can immediately start playing by:

    - Selecting a game mode (e.g., Normal Mode or Timed Mode).
    - Tracking their progress, high scores, and overall accuracy.

## âœ¨ Features

### 1. The Core Quiz Game
- **Multiple Categories:** Test your knowledge in Geography, Science, History, Pop Culture, Literature, and more.
- **Game Modes:** Choose between a relaxed "Normal Mode" (no time limit) or a challenging "Timed Mode."
- **Progress Tracking:** An integrated stats panel shows your total quizzes played, overall accuracy, and highest score.
- **User Accounts:** A "Sign in / Register" feature allows users to save their progress.

### 2. The Study App Suite

The app dashboard provides one-click access to a range of productivity tools:
- **Notes:** A simple note-taking app.
- **Flashcards:** Create and review digital flashcards.
- **Calculator:** A basic calculator for quick math.
- **Study Timer:** A timer to manage study sessions (e.g., Pomodoro technique).
- **Dictionary:** An in-app dictionary for quick lookups.
- **To-Do List:** Manage your tasks and stay organized.
- **Quiz Game:** A link to return to the main quiz app.

## ðŸš€ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing.

### Prerequisites

You will need to have Node.js (v18 or later) and pnpm installed on your system.

### Installation & Setup

1. Clone the repository:

  ```bash
  git clone [https://github.com/mariranasinghe/QuizApp.git](https://github.com/mariranasinghe/QuizApp.git)
  ```


2. Navigate to the project directory:

  ```bash
  cd QuizApp
  ```


3. Install dependencies: This project uses ```pnpm```. To install the required packages, run:

  ```bash 
  pnpm install
  ```


4. Run the development server: Once the dependencies are installed, you can start the app in development mode:

  ```bash 
  pnpm dev
  ```


5. Open the app: Open ```http://localhost:3000``` with your browser to see the application.
